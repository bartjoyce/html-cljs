Samples

  • (first list)
    Returns the first element of a given list.
  • (rest list)
    Returns what remains of the list (after removing the first element).
  • (cons element list)
    Returns a list with element concatenated to the front.
  • (eq a b)
    Returns whether or not a and b are equal.
  • (typeof expr)
    Returns the type of a given expression.
  • (cond (cond1 expr1) (cond2 expr2))
    Returns the expression for which the condition is non-nil.
  • (fn [params] function-body)
    Returns a lambda function which takes params as symbols and executes its function body. The function body can consist of any number of expressions; the value of the last one will be returned.
  • (def symbol expr)
    Assigns the value of expr to a given symbol in the public scope. It then returns the value of expr.
